## Featured Project

### [SaaS and Tech-Enabled Services Valuation Model](https://github.com/EvanJReid/SaaS-Valuation-Tool)

An institutional-grade, browser-based valuation tool for evaluating SaaS companies. Built to replicate the analytical frameworks used in private equity and M&A diligence processes, with all multiples and benchmarks sourced from primary market data.

<br/>

| Module | Description |
|:---|:---|
| **EV/ARR Multiple Build-Up** | 9 source-cited adjustment factors in a waterfall (Rule of 40, NRR, GRR, LTV:CAC, size premium) |
| **Full DCF Schedule** | Revenue through NOPAT to Free Cash Flow with margin expansion |
| **ARR Bridge** | New logo + expansion − contraction − churn with NRR reconciliation check |
| **Cohort Retention Curves** | 5 vintage cohorts modeled over 7 years |
| **Sensitivity Heatmaps** | Full model recomputed at every cell — no approximations |
| **Diligence Scorecard** | 20-item weighted PE-standard checklist |

<br/>

> All benchmarks sourced from SEG, SaaS Capital, Aventis Advisors, and Benchmarkit.
