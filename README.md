## Featured Project

### [SaaS and Tech-Enabled Services Valuation Model](https://github.com/EvanJReid/SaaS-Valuation-Tool)

> An institutional-grade, browser-based valuation tool built to replicate the analytical rigor of private equity and M&A diligence processes. All multiples and benchmarks are sourced from primary transaction data — not estimates.

<br/>

**Valuation Methodology**

| Component | Detail |
|:---|:---|
| Primary Method | EV/ARR multiple build-up with 9 source-cited adjustment factors |
| Rule of 40 Adjustment | +2.2x per 10 points — sourced exact from Aventis Advisors (459 deals, 2015–2025) |
| NRR Adjustment | Calibrated to SEG 2025 Annual Report p.30: <90% = 1.2x, 100–110% = 6.0x, >120% = 11.7x |
| Bear / Bull Scenarios | Anchored to Windsor Drake M&A 2025 transaction data (not arbitrary discounts) |
| DCF Model | Full schedule: Revenue → COGS → GP → OpEx → EBITDA → NOPAT → FCF |
| Sensitivity Analysis | Full model recomputed at every heatmap cell — no cell-level approximations |

<br/>

**Analytical Modules**

| Module | Description |
|:---|:---|
| ARR Bridge | First-principles waterfall: new logo + expansion − contraction − churn, with NRR reconciliation check |
| Cohort Analysis | 5 vintage cohorts modeled over 7 years using logo churn and per-customer NRR expansion |
| Comparable Analysis | Side-by-side comp table with best-in-peer highlighting across 8 metrics |
| Diligence Scorecard | 20-item weighted checklist built to PE lender covenant standards |

<br/>

**Data Sources**

| Source | Data Used |
|:---|:---|
| Software Equity Group (SEG) | 3,163 private M&A deals (2024) — transaction medians and NRR band data |
| SaaS Capital | Annual survey n=1,000+ — private company multiple benchmarks |
| Aventis Advisors | 459 deals (2015–2025) — Rule of 40 regression coefficient |
| Benchmarkit | n=936 (2024) — LTV:CAC median, Magic Number, CAC payback |
| Windsor Drake M&A | 2025 — bear/bull scenario discount/premium anchors |
| ABF Journal | 2025 — GRR lender covenant floor (85%) |
